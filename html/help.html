AIDE

Ce cite à pour but d'aider les reseaux sociaux a transmettre les informations lors de situations d'urgence

l'onglet accueil donne les derniers informations a savoir sur les situations urgente récente
elle possede egalement une carte indiquant les zones dans lesquel une situation d'urgence à ete signialer

la page de connection permet de s'identifier sur le site et permet par la suite de poster de message ou de signialer un nouvelle evenements

la page d'enregistrement permet de cree un compte sur le site

la page de creation d'evenement permet de signialer un evenement dans la base
