AIDE

Ce site a pour but d'aider les reseaux sociaux a transmettre les informations lors de situations d'urgence.

L'onglet accueil donne les dernières informations à savoir sur les situations urgentes récentes
elle possède egalement une carte indiquant les zones dans lesquelles une situation d'urgence à été signalé.

La page de connexion permet de s'identifier sur le site et permet par la suite de poster des messages ou de signaler un nouvel evenement.

La page d'enregistrement permet de créer un compte sur le site.

La page de création d'évènement permet de signaler un évenement dans la base.
